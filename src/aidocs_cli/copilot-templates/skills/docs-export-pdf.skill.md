# Documentation PDF Export Skill

Export markdown documentation to PDF with table of contents.

## What this skill does:

1. Converts markdown documentation to PDF
2. Generates table of contents
3. Applies consistent styling
4. Creates professional-looking documentation

## Instructions:

Load and execute the workflow from: `.copilot/workflows/docs/export-pdf/workflow.md` in the project repository.

Follow the workflow instructions exactly as specified in that file.

## Usage Patterns:

The user might say:
- "export docs to PDF"
- "create PDF documentation"
- "generate PDF from markdown"
- "export documentation as PDF"

## Features:

- Automatic table of contents
- Syntax highlighting for code blocks
- Image embedding
- Link preservation
- Page numbering

## Output:

- Professional PDF documentation
- Bookmarks for navigation
- Preserved formatting
- Embedded images
